Projeto Biblioteca Bates

# biblioteca_bates

1. Crie seu ambiente virtual:

```Git bash
python -m venv venv
```

2. Ative seu venv:

```bash
# linux:
source venv/Scripts/activate
```

## Instalação dependencias

**`pip install -r requirements.txt`**

## atualizando as Instalação dependencias

**`pip freeze > requirements.txt`**
